# Aider AI Context - LeetCode Tracker

## Project

**LeetCode Progress Tracker** - Serverless app for tracking coding interview practice.

## Tech Stack

- **Frontend:** React 19 + Vite 7 + TypeScript
- **Backend:** AWS Lambda (Node.js 20 + TypeScript)
- **Infrastructure:** AWS CDK v2
- **Database:** DynamoDB (single-table design)
- **Auth:** AWS Cognito (JWT)
- **Validation:** Zod v3.23.8
- **Monorepo:** Turborepo + pnpm 9.0.0

## Commands

```bash
pnpm install              # Install deps
pnpm dev                  # Dev mode
pnpm build                # Build all
cd infra && npx cdk deploy --all  # Deploy
```

## Workspaces

- `@leetcode-tracker/backend` - Lambda functions
- `@leetcode-tracker/web` - React frontend
- `@leetcode-tracker/infra` - CDK infrastructure
- `@leetcode-tracker/shared-types` - Shared types

## Rules

1. Use TypeScript strictly
2. Validate all inputs with Zod
3. Use AWS SDK v3 (modular)
4. Never commit credentials
5. Test CDK with `cdk diff`

## API

**Base:** `https://<API_ID>.execute-api.us-east-1.amazonaws.com/<STAGE>/`
**Auth:** `Authorization: Bearer <JWT_TOKEN>`

Endpoints:

- POST /trackers
- GET /trackers
- GET /trackers/{id}
- PUT /trackers/{id}
- DELETE /trackers/{id}
